# BharatScan v1.1.0 — Release Notes

## Highlights
- Tricolor Modern UI refresh with India-inspired visuals, improved spacing, and clearer hierarchy.
- Background image support (Home, Documents, Settings) with tint overlays for better readability.
- OCR Search Index: search across scanned text within documents.
- Guided walkthrough: in-app tutorial flow across Home ? Scan ? Edit ? Export ? Search.
- Update experience improvements: GitHub Releases checks, progress UI, release notes display, and startup checks.

## UI/UX Improvements
- Saffron primary actions, refined navy accents, and softer surfaces for stronger separation from backgrounds.
- Enhanced Home hero styling and card polish.
- Documents and Search screens now use layered surfaces for inputs and headers.
- Settings layout compacted with clearer grouping and improved header readability.

## Scanner & Editing
- Better edge-detection stability and fallback to manual crop.
- Retake action available per captured image.
- Crop grid refinements for manual adjustment.

## Localization & Typography
- Improved font fallbacks for Indian scripts to avoid missing glyphs.
- Polished localized strings and branding consistency.

## Performance & Reliability
- OCR text persisted per document to enable faster search and future indexing.
- Background rendering optimized for image + tint + scaling.
